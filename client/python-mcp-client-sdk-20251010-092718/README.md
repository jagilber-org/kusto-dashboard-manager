# Python MCP Client SDK Package

Exported: 2025-10-10 09:27:18

## Installation

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run a client:
   ```
   python working-py-mcp-client.py
   python simple-py-mcp-client.py
   python basic-py-mcp-client.py
   ```

## Files Included

- working-py-mcp-client.py  - Full-featured MCP client
- simple-py-mcp-client.py   - Simplified MCP client
- basic-py-mcp-client.py    - Basic MCP client
- py-mcp-client.py          - Minimal client stub
- requirements.txt          - Python dependencies
