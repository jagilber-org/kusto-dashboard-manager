# Dashboard Snapshots Directory

This directory contains accessibility tree snapshots captured from Azure Data Explorer dashboards page.

## ⚠️ Privacy Notice

**Files in this directory are excluded from git** because they contain:
- User names (dashboard creators)
- Dashboard names (may be internal/confidential)
- Dashboard IDs
- Access timestamps
- Account information

## Contents

Typical files:
- `dashboards-list.yaml` - Snapshot of all dashboards list page
- `dashboard-{id}.yaml` - Individual dashboard snapshots
- `dashboard-structure-analysis.md` - Analysis notes (safe to commit if anonymized)

## Usage

These snapshots are generated by:
1. Playwright MCP browser automation navigating to dashboards
2. `browser_snapshot` tool capturing accessibility tree
3. Saved locally for parsing and analysis

## Security

- **Never commit** actual snapshot YAML files
- **Do commit** example/template files with PII removed
- **Do commit** documentation and analysis with generic examples
